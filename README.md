### 一、相关技术
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 后端：`Java`、`JavaWeb` / `Springboot`等。
- 数据库：`MySQL` / `SQLServer` /等

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等
### 三、部署所需环境

- 后端：`jdk1.8`。
- 前端：`node`、`vue-cli` 、`npm`  。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示
系统分为：用户、管理员。

用户功能：登录、注册、首页、动漫信息、社区互动、公告信息和在线留言等功能

管理员功能：登录、个人中心、用户管理动漫分类、动漫信息管理、订阅信息管理、更新通知管理、在线留言、社区互动和系统管理等功能
